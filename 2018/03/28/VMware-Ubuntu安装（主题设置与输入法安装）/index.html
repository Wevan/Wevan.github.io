VMware+Ubuntu安装（主题设置与输入法安装）

发表于 2018-03-28 | | 阅读次数:

安装

安装VMware的WorkStation Pro（百度软件的亦可），Ubuntu

在页面底部选择Not now, take me to the download ›

启动VMware安装包，安装完成后启动，添加虚拟机，选择刚刚下载的Ubuntu的iso。添加虚拟机，一路默认到最后，启动即可。
需要进行说明的是：

  • 安装包一定要以管理员身份运行，否则会在最后一步报出没有权限执行的问题。

  • 设置用户名时不要使用root字样，修改为自己其他的用户名

安装完成后等待些许即可启动，输入密码即可

此处建议开机以前先编辑虚拟机，将虚拟机内存调整至2G以上，否则运行会十分卡顿

安装输入法

默认火狐浏览器，打开百度输入法即可，我这里安装的是搜狗输入法。下载Linux版安装包。

打开安装包所在位置，在该文件夹下打开终端

执行

sudo dpkg -i 安装包名

系统设置–>Laguage Support，一开始的系统里面是没有小企鹅输入法的，搜狗输入法就是在小企鹅输入法基础上安装的，从语言支持里面选择小企鹅输入法（fcitx），默认为ibus

有时候会出现提示你Software database is broken，这时候终端使用

sudo apt-get install -f

即可

系统会提示你安装language support，Install即可

安装完成后，把系统键盘输入方式从ibus切换为fcitx

终端执行

sudo reboot

完成重启电脑。

重启完成后一般屏幕右上方有了企鹅输入fcitx，如果你是先安装了主题，那么看到的就不是小企鹅，而是下图右顶端图标中的这个A字的块

右键选择Configure,点击+,

搜索Sougou pinyin,添加搜狗输入法,选中,OK

然后再次终端执行

sudo reboot

重启完成后即可在小企鹅或者A字块图标下选择Sougou输入法输入中文

美化主题

准备工作

首先在Settings里将Software&Updates里下载自修改为Others–>aliyun服务

在此处你需要将所有软件更新到最新,右键Terminal终端执行

sudo apt-get update
sudo apt-get upgrade

Linux下键入密码是不会像Windows下那样显示一串**的，你输什么它都不会显示，输就好了，也是为了安全起见

如果执行失败报错Could not…，执行

ps -ef

找出带apt的进程，执行

sudo kill -9 进程号

然后执行

sudo apt-get install unity-tweak-tool

安装主题

首先安装主题

sudo add-apt-repository ppa:noobslab/themes
sudo apt-get update
sudo apt-get install flatabulous-theme

安装配套图标

sudo add-apt-repository ppa:noobslab/icons
sudo apt-get update
sudo apt-get install ultra-flat-icons

安装完成后，左上角图标打开搜索unity-tweak-tool，修改主题和图标：

Theme，修改为Flatabulous

icon修改为Ultra-flat
